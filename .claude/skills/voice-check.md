# Voice Check

Review any piece of content against Marina's voice DNA before it goes out. Return a clear verdict and specific edits — not vague feedback.

## How to run
Take the content Marina shares and run it against the Voice DNA profile. Score it across four dimensions:

**1. Rhythm** — Does it flow like Marina speaks? Or does it have an AI choppiness, a listicle structure, an unnatural cadence?
**2. Vocabulary** — Any banned words? Any hollow phrases? Does it use her vocabulary or generic marketing language?
**3. Philosophy** — Does Marina's worldview come through? Connection, depth, freedom, real over transactional — is it present or absent?
**4. Authority** — Does it sound like someone who genuinely knows Bali's rooms and its people? Or does it sound like a copywriter guessing?

## Output format
Return a **Voice Report Card:**

| Dimension | Score (1–5) | Notes |
|---|---|---|
| Rhythm | | |
| Vocabulary | | |
| Philosophy | | |
| Authority | | |

**Overall verdict:** PUBLISH / REVISE / REWRITE

Then provide the revised version — not notes on what to fix, the actual fixed version. Marina should be able to copy-paste the output directly.

## Rules
- Never say "great job" or soften the verdict. Be direct.
- If the piece passes on all four dimensions, say so clearly and quickly.
- If it fails, rewrite it — don't just flag it.
- The goal is zero AI tells in published content.
